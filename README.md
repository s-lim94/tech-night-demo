# tech-night-demo

A Claude Code [plugin marketplace](https://docs.claude.com/en/docs/claude-code/plugins) hosting demo plugins for Tech Night.

## Install

```
    /plugin marketplace add xsolla/tech-night-demo
    /plugin install qa-webshop-audit@tech-night-demo
```

## Plugins

### `qa-webshop-audit`

Browser-driven QA audit: reads a YAML spec, drives Chromium against a target site, captures screenshots, and produces a structured QA report. Designed for live demos.

See [`plugins/qa-webshop-audit/skills/qa-webshop-audit/SKILL.md`](plugins/qa-webshop-audit/skills/qa-webshop-audit/SKILL.md) for details.
